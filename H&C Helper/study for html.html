HTML

Introduction:

When you send request to a web server through a web browser to access a webpage, you get HTML as a response from there.The web browser understands the HTML and displays contents of the webpage.HTML tells the browser how the contents are structured inside a webpage.

Definition

Hypertext Markup Language (HTML) is a simple language to create webpages.
There are two important terms that you need to understand in the name HTML.

      1- Hypertext
      2- Markup Language

Hypertext

The term Hypertext is used due to the special text in a wepage called hyperlinks. By clicking on these links you can move from one webpage to another.Hyperlinks are used to navigate on the World Wide Web (WWW).

Markup Language

A webpage consist of a series of element which are represented by tags.For example, if you need a paragraph of text on your webpage,you use it as:
        <p> Some Text Here <p>
Here <p> shows making of paragraph opening tag and </p> means marking of paragraph closing tag.Due to marking of each element,it is called a markup language.

         <p> I am a student </p>
         <p> I am in class 9 </p>
         <p> I am in computer section </p>
         <p> I have a computer </p>

Identifying the tags Used to Markup HTML Elements

There are two tags in an HTML document.

  1. Paired Tags
  2. Singular Tags

Paired Tags

Most of the tags in HTML are paired tags.They consist of a start tag,an end tag and contents between them. Following is the general structure of paired tags.

                <tagname></tagname>
For example, tag p to create a paragraph in HTML document is a paired tag.

                <p> I am in 9th class. </p>

Singular Tags

Some tags do not have closing tags and they are called singular tags or empty tags.They are simply written as <tagname>. For example, <br> for line break, <hr> to insert a horizontal line.

Attributes in HTML Tags

Attributes are the properties association with tags.They provideo some information respects to a specific tag.Each attributes given a value Generally, a tag with attribute is written as:

<tagname attribute1="value" attribute 2="value"......attributeN="value"><tagname>
